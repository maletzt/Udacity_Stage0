Introduction to the Web

The Web

  • Collection of HTML (HyperText Markup Language) Documents.
  • Main type of document on the web are HTML. They are the basis for everything that is on the web.
  • The types of files that can be found on the web are
    • Plain Text
    • HTML
    • Images
    • Videos
    • Music
  • Major Pieces (Concepts)
    • You, Your Computer, and Browser
    • The Internet
    • Servers

HTML

  • HTML is made up of text content (what you see)
  • Markup (what is looks like)
  • References to other documents (images, videos, etc.)
  • Links to other pages
  • HTML Markup consists of <NAME> Contents </NAME> or Opening tag + Contents + Closing Tag. The whole things is referred to as the Element.
    • The b Tag is for bold (example <b> troy </b>.
    • The em - emphasis Tag is for italics (example <em> troy </em>)

HTML Attributes

  • Tags can have attributes (example <TAG ATTR="value">contents</tag>). Note tags can have mulitple attributes
  • Anchor tags <a> are used to make links (example <a href="www.reddit.com"> derp </a>)
  • Image tags <img> are used to insert images into HTML (example <img src="http://www.udacity.com/cs253xhipmunk.png" alt="hipmunk!">) Note alt text is for broken url. Also there is no closing tag for images. Image Tags have a Void Tag (No content is considered a Void Tag).
  • To put text on mulitple lines you can use a line break <br> or a paragraph <p></p>

Inline versus Block

  • The <br>, <em>, <span></span> and <img> tags are examples of inline
  • <p> and <div></div> tags are an example of block. A block creates and invisible block that can have height and weight versus inline which is considered just text.

Recap

The web is composed largely of HTML documents. HTML stands for Hypertext Markup Languageand is the language the browsers use to interpret web pages. The components of the web include the client (your computer and browser), the Internet, and servers. The client and servers interact through something called the Hypertext Transfer Protocol (HTTP).

HTML Elements are what we use to tell a web browser how to display content in a web page. Most HTML elements consist of an opening and a closing tag with some content between them
